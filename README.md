# IoT Using OTP & Fingerprint Based Door Lock System

## Project Description
This project implements a smart door lock system using
Fingerprint authentication and One-Time Password (OTP).
The system uses ESP32 with IoT capabilities for secure access control.

## Technologies Used
- ESP32 Microcontroller
- Fingerprint Sensor
- OTP Generation
- Telegram Bot (IoT)
- Embedded C (Arduino IDE)

## Working
1. Fingerprint verification
2. OTP generation and delivery via Telegram
3. OTP entry through keypad
4. Door unlock using relay and solenoid lock

## Applications
- Smart homes
- Offices
- Restricted areas
